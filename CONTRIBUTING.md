# Contributing

Contributions are released under CC0 1.0 Universal / public domain dedication unless a file explicitly says otherwise.

Hermes/zuko contributions are authored as `hermezuko` and are intended to be public-domain/CC0 contributions.

## Generation acknowledgement

Contributors should expect that code and project materials may be generated, edited, or assisted by Hermes Agent and backing AI model(s). This should be acknowledged transparently in project documentation without adding restrictions to the CC0/public-domain dedication.
